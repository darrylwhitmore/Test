Remarks
The Close method closes the binary response stream and releases the connection to the resource for reuse by other requests.

You must call either the Stream.Close or the BinaryWebResponse.Close method to close the stream and release the connection for reuse. It is not necessary to call both Stream.Close and BinaryWebResponse.Close, but doing so does not cause an error. Failure to close the stream can cause your application to run out of connections.
